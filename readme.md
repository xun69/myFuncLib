# myFucLib

## 概述

从3.5开始一直在积累GDScript的静态函数库，方便在应用程序和日常游戏项目、插件开发中使用。

本仓库就是专门将这部分内容拉出来独立作为一个Godot项目进行维护，并同步维护相关的文档。
